1. What is a module?
    A file containing Python code (functions, variables) that you can reuse in other files.

2. List three ways to import a module in Python.
    a. import whole module
    b. import a specific function
    c. import with an alias 

3. What is the purpose of importing?
    For efficieny: To reuse code without rewriting it — either from Python's built-in library or third-party packages.

4. List three examples when you would use the random module.
    a. random.randint(1, 10)      # random number between 1-10
    b. random.choice([1,2,3])     # random item from a list
    c. random.shuffle([1,2,3])    # shuffle a list randomly

5. What is an ImportError?
    An error that occurs when Python can't find the module you're trying to import, e.g. it's not installed or the name is misspelled.

6. When would using an OrderedDict be useful?
    An OrderedDict may be useful when you need to compare two dicts and their order matters. 

7. When would using a defaultdict be useful?
    When you want a dictionary that automatically creates a default value for missing keys instead of throwing a KeyError.

8. What is the purpose of the following code:
    if __name__ == '__main__':
    pass

    It means "only run this code if this file is being run directly, not when it's imported as a module by another file." pass here is just a placeholder meaning "do nothing".
